# jsoninf

JSON schema inference, especially from JSON lines.

# Problem

Given a blob of JSON documents w/o explicit schema, infer structure and data
types. This tool can be used to detect structural and typing issue in large
JSON datasets.

# Usage

```shell
$ jsoninf < file.ndj
```
